## Getting Started

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd dessert_app
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) to view the app in your browser.
